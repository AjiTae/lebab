Usage


        

            xto6 transpiles your ES5 code to ES6. It does exactly the opposite of what other
            transpilers do (like babel and traceur).
        


        
Install it using npm :

        
$ npm install -g xto6


        
Transpile your boring code using the xto6 cli tool.


        
xto6 es5.js -o es6.js
