Getting Started

xto6 transpiles ES5 code into ES6 code. It acts like Babel but in reverse order! (you can call it lebab if you like)

Install and use xto6 CLI using npm:

        npm install -g xto6

xto6 es5.js -o es6.js

Currently these ES6 features are supported

  • Classes
  • Arrow functions
  • String Templates
  • Modules (soon)
  • let, const (soon)
  • for of (soon)
  • You tell us!

Try it now!

        
var someClass = function () {
    console.log('This is the constructor.');
};

someClass.prototype.someOuterMethod = someFunction;

someClass.prototype.someInnerMethod = function (s, t) {
    var result = s + t;
    console.log(result);
    return result;
};

Object.defineProperty(someClass.prototype, 'someAccessor', {
    get: function () {
      return this._some;
    },
    set: function (value) {
      this._some = value;
    }
});

function someFunction(a, b) {
    return a + b;
}
